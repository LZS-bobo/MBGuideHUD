# MBGuideHUD
HUD for novice (新手引导页，非常简单的帮你创建需要高亮区域 ，拥有不同的形状)
# 用法
## 自定义
```
[MBGuideHUD showText:@"只因和我的心和你一起碎" visible:_btn];

```
